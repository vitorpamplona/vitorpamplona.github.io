nVidia GeForce 9600 GT e Ubuntu 8.04

Para quem está com problemas com o driver da nova nVidia 9600 para linux basta instalar o driver (> 173) diretamente do site dela, seguindo os passos abaixo.

Remova tudo relativo a nVidia que pode estar instalado via apt-get: 

sudo apt-get remove nvidia-settings nvidia-new-kernel-source 
                    nvidia-kernel-common nvidia-glx-new nvidia-cg-toolkit

Nesta hora ele pedirá para remover também os pacotes: 

linux-generic
linux-restricted-modules-generic

Você precisa aceitar. A remoção deles não fará mal ao sistema, apenas o Ubuntu não irá mais controlar os drivers restritos, como uma Webcam, por exemplo.

A seguir, instale os pacotes para a compilação do driver. 

sudo apt-get install build-essential libc6-dev

Baixe uma versão do driver igual ou superior a 173.08 a partir do site: ftp://download.nvidia.com/XFree86/Linux-x86

UPDATE: O pacote de drivers Envy no Ubuntu 8.04 já tem o driver da 173. Pode-se instalar esse pacote em troca dos passos abaixo. Pelo envy, na hora de instalar os drivers, selecione nVidia, instalação Manual e aí o driver 173. 

wget ftp://download.nvidia.com/XFree86/Linux-x86/173.08/NVIDIA-Linux-x86-173.08-pkg1.run 
chmod 777 NVIDIA-Linux-x86-173.08-pkg1.run

Saia do X (Ctrl + Alt + F1) e logue-se como root (ou logue-se com o seu usuário e torne-se root através do comando su).

Derrube o GDM ou KDM: 

killall gdm 

e execute o arquivo baixado: 

sh NVIDIA-Linux-x86-173.08-pkg1.run

Basta seguir as instruções (Sim para todas as opções), reiniciar e pronto!
