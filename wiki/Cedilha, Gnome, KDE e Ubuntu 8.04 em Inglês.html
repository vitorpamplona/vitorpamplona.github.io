Cedilha, Gnome, KDE e Ubuntu 8.04 em Inglês

O cedilha não funciona ao utilizar o Gnome ou KDE no Ubuntu em inglês com um teclado americano. Este post corrige o problema.

Vale lembrar que se a língua padrão do sistema for alterada para português do brasil, o Cedilha funcionará automaticamente. Para tal, vá no menu System / Administration / Languages Support e mude o padrão para português. Mas, se você é como eu que quer manter o inglês, siga os passos abaixo.

Selecione o layout de teclado chamado USA Alternative International (former us_intl) em System / Preferences / Keyboard / Layouts.

Edite o arquivo um destes arquivos (ou todos): 

sudo vim /usr/lib/gtk-2.0/2.10.0/immodule-files.d/libgtk2.0-0.immodules
sudo vim /usr/lib/gtk-2.0/2.10.0/gtk.immodules 
sudo vim /usr/lib/gtk-3.0/3.0.0/immodules.cache

Procure uma linha que começa com cedilla e adicione um : en no último parâmetro, assim: 

"cedilla" "Cedilla" "gtk20" "/usr/share/locale" "az:ca:co:fr:gv:oc:pt:sq:tr:wa:en"

Salve e saia.

Neste ponto o Ç deve funcionar nas aplicações GTK. Agora vamos ajustar as aplicações do QT.

Edite o arquivo Compose 

sudo kate /usr/share/X11/locale/en_US.UTF-8/Compose

Procure por todos as instâncias de ć e troque por ç e do Ć para Ç . Se vc não conseguir digitar o cedilha, copie e cole daqui mesmo.  

Para o KDE4, é possível fazer o cedilha com AltGt + ", " e C. Se o seu teclado não possui um AltGr, basta configurar o alt direito para se comportar como AltGr no nas opções de sistema.  

Reinicie e pronto.
